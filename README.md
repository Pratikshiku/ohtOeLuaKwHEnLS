## Student Management System Course Selection Attendance Leave

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/00c305229f9e4e22a5e6be62aa16ee33/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Navicat`
- `Visual Studio Code(VScode)`
- etc.

### 3. Function Description
The system is divided into (roles): Student, Teacher, Administrator.

**Student Functions:**
1. Login
2. Student Information Management (Personal Information)
3. Course Selection Information
4. Attendance Information
5. Leave Information
6. Grade Information
7. Change Password
8. Secure Logout

**Teacher Functions:**
1. Login
2. Student Information Management
3. Class Information Management
4. Teacher Information (Personal Information)
5. Course Information Management
6. Course Selection Management
7. Attendance Management
8. Leave Management
9. Grade Management
10. System Management
11. Change Password
12. Logout

**Administrator Functions:**
1. Login
2. Student Information Management
3. Class Information Management
4. Teacher Information
5. Course Information Management
6. Course Selection Management
7. Attendance Management
8. Leave Management
9. Grade Management
10. System Management
11. Change Password
12. Logout
